# MyDJ Portal README
Welcome to the MyDJ Portal git repository. This repo contains the API and Front-end for the MyDJ Portal system, which is a all-in-one solution for financing, gig administration and interaction with your clients. 
The API is written in Django (Python) and requires a Apache webserver with WGCI module to run and a MySQL database. The front-end is written in Angular and runs on any webserver.

## Invoice Ninja v5 required
Please note that this system is build to interact with the self-hosted API of Invoice Ninja. This is a (free) PHP invoicing system to generate invoices and quotes. The clients, quotes and invoices of the system relies on a active installation.
